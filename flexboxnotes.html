Flexbox Notes

Flexbox is used for formatting elements inside of a container. It only works for elements that are directly inside the container (not nested within another element in the container)

CSS Styling Syntax:

Setup: in the styling section for the overarching container, put "display: flex"

Flex-direction:

  • row: horizontal row (default)
  • column: vertical column
  • row-reverse/column-reverse: swaps order of elements

Flex-wrap: controls if elements go onto next line (if they don't fit on one line)

  • wrap: elements will go onto next line (default)
  • nowrap: if elements don't fit on one line, they will be squeezed so that they fit
  • wrap-reverse: elements start from bottom and wrap upwards

Justify-content: defines alignment along main axis

Align-content: defines aligment along the cross axis

Gap: row and column gap between items (gap: 50px 100px)

Order: goes in the styling section for each individual element, defines the order that the elements are in (ex: order: 3)

Flex-basis: sets the initial main size of an element (has to be at least this big)

Flex-grow: allows an element to grow to fill leftover space, works well for resizing browsers

Flex-shrink: allows an element to shrink to fit the content inside of it

Align-items: aligns items on main axis

  • If you want individual items to be exempt from an align-item rule, you can add "align-self" to the individual item's style section
