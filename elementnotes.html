Elements: defined by a start tag, some content, and sometimes a closing tag

Common Elements
  • Paragraph: p, separates a chunk of text
  • Headings: h1, h2...h6, from largest to smallest
  • Images: inserts an image from a folder in the root folder (img src="link")
  • Links: hyperlink to another site/page (a href="link")
  • Lists: can be ordered (ol) or unordered (ul)
Section Elements (Containers):
  • header: defines a header section
  • section: defines a section in the document
  • article: defines independent, self-contained content
  • aside: defines content aside from the content
  • footer: defines a footer for a document or a section
  • details: defines additional details that the user can open and close on demand
  • summary: defines a heading for the details element
  • nav: defines a set of navigation links
  • main: contains main content for the site
  • div: defines a division in the HTML file, and groups HTML elements so they can be styled all together with CSS. Not specific though! Other tags preferred
  • span: similar to a div tag except used for inline organization and styling, while a div tag is for block-level organization
